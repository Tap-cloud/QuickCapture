# Cove

Your personal corner of the internet. Save anything — YouTube videos, articles, recipes, notes — and find them instantly with smart search.

## Web App

[Open Cove](https://quick-capture-delta.vercel.app)

## Chrome Extension

Save any webpage to Cove in one click directly from your browser.

### Install

1. Download or clone this repository
2. Open Chrome and go to `chrome://extensions`
3. Enable **Developer mode** (toggle in the top right)
4. Click **Load unpacked**
5. Select the `extension` folder from this repo
6. The Cove icon will appear in your Chrome toolbar

### Usage

1. Sign in with your Cove account (same email/password as the web app)
2. Browse to any page and click the Cove icon to save it
3. Add an optional note before saving
4. Your save will appear instantly in the Cove web app
